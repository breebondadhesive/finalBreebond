# **App Name**: BreeBond Portfolio

## Core Features:

- Hero Section Animation: Parallax background animation with the BREEBOND logo rising from the bottom on scroll.
- Product Showcase Gallery: Horizontal scroll-snap gallery to showcase BREEBOND's product range (TYPE-2 101, TYPE-2, TYPE-3, TYPE-4) with parallax depth and 3D flip effect on hover.
- Counter Animations: Count-up animation for statistics like 'Years of experience' in the 'Why Choose BREEBOND' section.
- Technical Specification Accordion: Accordion component to display product specifications with animated data reveal on expand/collapse.
- Contact Form: Contact form in the footer with animated input focus states.

## Style Guidelines:

- Primary color: Dark Blue (#1A237E) to evoke trust and professionalism.
- Background color: Very light desaturated blue (#E8EAF6) to provide a subtle, clean backdrop.
- Accent color: Gray (#757575) to highlight product variants, providing a modern touch.
- Headings: 'Poppins' (sans-serif) for a bold, modern, and precise feel.
- Body: 'PT Sans' (sans-serif) for a clean and readable experience.
- Technical Specifications: 'Source Code Pro' (monospace) for displaying data tables and code snippets clearly.
- Use CSS Grid and Flexbox for a fully responsive layout across different screen sizes. Breakpoints: Mobile (320px-768px), Tablet (769px-1024px), Desktop (1025px+).
- Implement scroll-triggered animations using Framer Motion, including parallax effects, fade-ins, slides, and scaling. Optimize performance by disabling parallax effects on mobile.
